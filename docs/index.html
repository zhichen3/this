About Me

I'm a Graduate student studying computational astrophysics at Stony Brook University. I'm currently working under Prof. Michael Zingale on studying astrophysical explosions X-ray burst and Type 1a supernovae using numerical simulations. The main code I use is Castro, which is an adaptive mesh, astrophysical radiation hydrodynamics simulation code that is completely open-source.
